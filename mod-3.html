Common HTML Elements

Adding CSS

While there are multiple ways to give html css properties, the one we will be focusing on as professional developers is linking a style sheet.

A seperate css file will be created with all the applicable propeties for a page. Inside the head of the html, a link tag will be used. The link will have the 'rel' attribute of 'stylesheet' and the 'href' of the location of the stylesheet. This can be local or hosted online.

link rel="stylesheet" href="./styles.css"

CSS Selectors

A selector is how we choose and modify an html element.

Type Selector

Selects the element by element name. Just the name of the element.

p {}

div {}

Class Selector

Selects the element by all matching classes. Prepended by a dot (.)

.purple-text {}

.profile-pic {}

ID Selector

Selects element by ID. Prepended by a hashtag (#)

#carosel {}

#nav-bar {}

Universal Selector

Selects ALL elements. Only the wildcard/asterik (*)

* {}

Attribute Selectos

Selects specific elements that have a certain attribute. The element name followed by the attribute in square brackets.

a[href] {}

img[src] {}

CSS Selector Combinations

We can chain together several different selectors to target very specific elements.

Multiple Elements

A comma between elements will add that element to a list that targets them all.

h1, h2, h3 {}

Compund Selectors

Naming two elements right after the other (with no space in-between) will select all elements A WITH the class of B.

.profile-pic.admin {}

Descendant Selectors

Naming an element after another with a space in-between will target all elements B that are any degree of children of A. They do not have to be directly connected. Essentially if B is contained within element A in any way.

.guild-card .card-text {}

Direct Child Selector

Naming an element after another with a '>' between them will target only the immidiate children B of parent A. Must be a direct connection.

,guild-card > img {}

Adjacent Children

Naming an element after another with '+' between will target all B elements that come right after A.

ul + li {}
